Little Alchemy 2

Start with four basic elements and combine them to create the entire world!

How to Play

Simply drag and drop elements from the sidebar onto the main screen. Drag one element on top of another to try and combine them. If the combination is valid, a new element is created!

About This Game

With a clean interface, beautiful illustrations, and hundreds of items to discover, this game is a journey of creativity. The game features a library that keeps track of all your discoveries and the combinations you used. Can you find them all?
